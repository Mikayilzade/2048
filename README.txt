2048 — установка на iPhone как отдельного офлайн-приложения

1. Загрузите все файлы из этой папки в корень сайта с HTTPS.
   Самый простой бесплатный вариант — GitHub Pages.
2. Один раз откройте опубликованный адрес в Safari при наличии интернета.
3. Нажмите «Поделиться» → «На экран Домой» → «Добавить».
4. Запускайте игру нажатием на значок 2048.
5. После первого открытия файлы игры сохраняются на устройстве, поэтому игра
   должна запускаться без интернета.

Важно:
- Не загружайте на сайт сам ZIP. Сначала распакуйте его и загрузите файлы.
- При обновлении игры замените файлы на сайте. Возможно, понадобится закрыть
  игру и открыть её ещё раз, чтобы загрузилась новая версия.
